# PPT 语音 AI 改写工具 打包部署详细文档（V1.1）

## 一、环境准备

### 1.1 系统要求
- **操作系统**：Windows 10 / Windows 11 64 位
- **Python**：3.10 64 位（[下载](https://www.python.org/downloads/)，安装时勾选 *Add Python to PATH*）

### 1.2 前置运行库（重要）
PyQt6 与 pywin32 依赖 **VC++ 2015-2022 x64 运行库**，缺失会导致打包后的 exe 闪退。

下载安装：https://aka.ms/vs/17/release/vc_redist.x64.exe

> 如果打包后运行闪退且无错误提示，**99% 是缺这个运行库**。

### 1.3 打包工具
```cmd
pip install pyinstaller==6.3.0
```

### 1.4 安装包工具
下载安装 [Inno Setup 6](https://jrsoftware.org/isdl.php)（制作 setup 安装程序）

## 二、项目文件准备

完整目录结构（不能缺失）：
```
PPTVoiceRewrite/
├─ main.py                 主程序源码
├─ generate_icon.py        图标生成脚本
├─ icon.ico                程序图标（256×256，已生成）
├─ requirements.txt        依赖清单
├─ build.bat               一键打包脚本
├─ install.iss             Inno Setup 打包脚本
├─ config/
│  └─ config.json          默认配置文件
├─ record/                 录音存储目录
├─ history/                历史记录目录
└─ docs/                   文档
```

> `record/` 和 `history/` 无需手动创建，程序启动时自动 `makedirs`。

## 三、依赖安装步骤

1. cmd 进入项目根文件夹：
   ```cmd
   cd PPTVoiceRewrite
   ```

2. 执行依赖安装：
   ```cmd
   pip install -r requirements.txt
   ```

3. **pyaudio 安装失败解决方案**：
   pip 安装 pyaudio 容易报错（需要 C 编译器），执行离线 whl 安装：
   ```cmd
   pip install PyAudio-0.2.13-cp310-cp310-win_amd64.whl
   ```
   > ⚠️ 注意：从文档复制命令时，连字符必须是 ASCII 的 `-`，不要用 Unicode 长破折号 `‐`。
   > whl 文件可从 https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio 下载。

4. 验证安装：
   ```cmd
   python -c "import PyQt6, pyaudio, requests, win32com, Crypto; print('全部依赖 OK')"
   ```

## 四、生成图标

如需重新生成图标（修改配色 / 样式）：
```cmd
python generate_icon.py
```
产物：`icon.ico`（多尺寸 16/32/48/64/128/256）+ `icon.png`

> 图标已随交付包提供，正常使用无需重新生成。

## 五、一键打包便携 EXE

### 5.1 执行打包
直接双击运行 `build.bat`。

脚本自动完成：
1. 检查 PyInstaller（缺失自动安装）
2. 检查 icon.ico（缺失自动生成）
3. 清理旧产物（build / dist / .spec）
4. PyInstaller 打包
5. 整理产物（复制 config / icon 到 dist）

### 5.2 打包参数说明

| 参数 | 作用 |
|---|---|
| `--onefile` / `-F` | 打包为单文件 exe |
| `--windowed` / `-w` | 运行不弹出控制台窗口 |
| `--name "PPTVoiceRewrite"` | 输出 exe 名称 |
| `--icon "icon.ico"` | 设置程序图标 |
| `--add-data "icon.ico;."` | 将图标打包进 exe（供 resource_path 读取） |
| `--collect-binaries pyaudio` | 收集 pyaudio 二进制（解决打包后运行失败） |
| `--hidden-import pyaudio` | 显式声明隐藏导入 |
| `--hidden-import win32com` | 显式声明 COM 组件 |
| `--hidden-import Crypto.Cipher.AES` | 显式声明加密模块 |

### 5.3 产物结构
```
dist/
└─ PPTVoiceRewrite/
   ├─ PPTVoiceRewrite.exe    便携版主程序
   ├─ icon.ico               图标
   ├─ config/
   │  └─ config.json         默认配置
   ├─ record/                录音目录（空）
   └─ history/               历史目录（空）
```

### 5.4 打包后验证
1. 进入 `dist\PPTVoiceRewrite\`
2. 双击 `PPTVoiceRewrite.exe`
3. 程序应正常启动，窗口标题为「PPT 语音 AI 改写工具 V1.1」
4. 关闭程序后，`config/`、`record/`、`history/` 应保留在 exe 同级目录

> ⚠️ 如果打包后运行闪退，去掉 `--windowed` 改用 `--console` 重新打包，保留黑窗口查看报错信息。

## 六、制作安装程序（setup 安装包）

### 6.1 编译安装包
1. 打开 Inno Setup Compiler
2. `File → Open` 选择项目根目录的 `install.iss`
3. 按 `Ctrl+F9`（或顶部 `Build → Compile`）编译
4. 编译完成后在 `安装包成品\` 目录生成：
   ```
   PPTVoiceRewrite_Setup_v1.1.exe
   ```

### 6.2 安装包特点
- ✅ 可选择安装目录
- ✅ 桌面快捷方式（可选）
- ✅ 开始菜单快捷方式 + 卸载入口
- ✅ 快速启动栏图标（可选）
- ✅ 升级安装保留 config / record / history（`onlyifdoesntexist` + `uninsneveruninstall`）
- ✅ 自带卸载程序
- ✅ LZMA2 Ultra64 压缩，体积优化
- ✅ 64 位专用安装

### 6.3 install.iss 关键配置

| 配置项 | 值 | 说明 |
|---|---|---|
| `DefaultDirName` | `{autopf}\PPT语音AI改写工具` | 默认安装到 Program Files |
| `ArchitecturesInstallIn64BitMode` | `x64compatible` | 64 位安装 |
| `PrivilegesRequired` | `admin` | 需管理员权限（COM 组件需要） |
| `Compression` | `lzma2/ultra64` | 最大压缩 |
| `SolidCompression` | `yes` | 整体压缩更小 |
| `UninstallDisplayIcon` | `{app}\PPTVoiceRewrite.exe` | 卸载列表显示图标 |

### 6.4 配置保留策略
```ini
[Files]
; config.json 仅首次安装写入，升级不覆盖
Source: "dist\PPTVoiceRewrite\config\config.json"; DestDir: "{app}\config"; Flags: onlyifdoesntexist

[Dirs]
; record/history 永不卸载删除
Name: "{app}\record"; Flags: uninsneveruninstall
Name: "{app}\history"; Flags: uninsneveruninstall
Name: "{app}\config"; Flags: uninsneveruninstall
```

## 七、打包常见问题解决

### 7.1 打包后运行闪退
**原因**：缺少依赖 / COM 组件问题 / 路径错误 / 缺 VC++ 运行库

**解决**：
1. 确认已安装 [VC++ 2015-2022 x64](https://aka.ms/vs/17/release/vc_redist.x64.exe)
2. 打包暂时去掉 `-w` 参数，保留黑窗口运行查看报错
3. 检查 `dist\PPTVoiceRewrite\` 下 config/record/history 目录是否存在

### 7.2 pyaudio 打包后运行失败
**解决**：
- `build.bat` 已包含 `--collect-binaries=pyaudio`，正常无需额外处理
- 如仍失败，追加 `--collect-all=pyaudio`

### 7.3 WPS 无法写入 PPT
**解决**：WPS 设置 → 高级设置 → 勾选「启用 COM 自动化」
> 必须是完整官方安装版，精简绿色版缺 COM 组件。

### 7.4 打包文件体积偏大
**原因**：PyQt6 库体积大（约 60-80MB）

**解决**：使用 upx 压缩
1. 下载 [upx](https://github.com/upx/upx/releases)
2. 解压 `upx.exe` 到 PyInstaller 同目录或 PATH
3. build.bat 追加参数：`--upx-dir "C:\path\to\upx"`
4. 压缩后体积可减少 30-50%

### 7.5 杀毒软件误报
PyInstaller 打包的 exe 容易被杀软误报。
**解决**：
- 加入杀软白名单
- 或使用代码签名证书签名（需购买证书）

### 7.6 图标不显示
**解决**：
- 确认 `icon.ico` 为多尺寸 ICO（generate_icon.py 已生成 16/32/48/64/128/256）
- Windows 资源管理器图标缓存可能需重启 explorer.exe 刷新

## 八、分发交付说明

### 8.1 便携版（绿色版）
1. 交付 `dist\PPTVoiceRewrite\` 整个文件夹
2. 用户复制到任意位置，双击 `PPTVoiceRewrite.exe` 即用
3. **无需安装 Python 环境**，但需 VC++ 运行库

### 8.2 安装版
1. 交付 `安装包成品\PPTVoiceRewrite_Setup_v1.1.exe`
2. 用户双击按向导完成安装
3. 桌面自动生成快捷方式
4. 升级直接覆盖安装，配置不丢失

### 8.3 首次运行
- 程序启动自动创建 `record/`、`history/` 目录（无需手动创建）
- 首次使用需点击「🔑 密钥设置」配置 OpenAI API Key
- 配置完成后自动后台校验密钥
